这个博客是如何诞生的？利用Hexo框架和GitHub搭建个人博客

安装Nodej以及hexo框架

配置_config.yml

在博客的根目录下有一个配置文件，最后一项部署项需要修改三个参数和自己的GitHub仓库对应

​ deploy:
type: git
repo: https://github.com/YourGithubName/YourGithubName.github.io.git
branch: master

hexo d #部署到Github仓库里
https://YourGithubName.github.io/ 通过这个URL访问这个地址可以查看博客

更换主题

  • 下载yilia主题到本地

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

  • 修改hexo根目录下的 _config.yml 文件 ： theme: yilia

hexo c #清理一下
hexo g #生成
hexo d #部署到远程Github仓库
https://YourGithubName.github.io/ #查看博客
